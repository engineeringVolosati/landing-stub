# 002 — GL assembly layout

Target: arrange the parts as a floating abstract construction, not a colliding swarm.

Scope:
- Build around a loose diagonal spine with asymmetry.
- Place about 30 parts in clusters: core, side ribs, small satellites.
- Preserve negative space so the shape stays architectural.
- Avoid intersections that look accidental; intentional overlaps must read as connected joints.

Done when:
- Parts positions are generated from a compact layout array.
- Desktop preview reads as a constructed object.
- Mobile preview keeps the silhouette inside the hero area.
